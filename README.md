# Meeting Intelligence System

A step-by-step ML/NLP project to build a meeting intelligence platform that can process meeting transcripts and extract useful insights such as action items, decisions, topic segments, and semantic search.

## Current Progress
- Step 1: Project setup
- Basic transcript cleaning pipeline

## Tech Stack
- Python
- spaCy
- scikit-learn
- sentence-transformers
- FAISS
- FastAPI
- Streamlit

## Project Goal
Build an end-to-end NLP + ML Engineer style system for meeting understanding.
